# DingsJuice Mobile App 🥤🇺🇬

Production-ready mobile application for fresh sugarcane juice delivery in Uganda.

## Tech Stack

- **Frontend:** React Native (Expo)
- **Styling:** NativeWind (Tailwind CSS)
- **Backend:** Firebase (Auth, Firestore)
- **Icons:** Lucide React Native
- **Payments:** Local Mobile Money (MTN MoMo, Airtel Money)

## Features

- Interactive Home Menu
- Secure Checkout with Mobile Money integration
- Order Tracking
- Real-time Inventory Management

## Local Development

1. Install dependencies:

   ```bash
   npm install
   ```

2. Start the development server:

   ```bash
   npx expo start
   ```

## Production Details

- **App Name:** DingsJuice
- **Package:** com.dingsjuice.app
- **Target Marketplace:** Google Play Store
